# Lucky Pharma Design System

**Версия:** 1.0
**Дата обновления:** 30.01.2026

Дизайн-система "Lucky Pharma" представляет собой набор стандартов, компонентов и стилей, используемых для создания консистентного и пользовательского интерфейса веб-приложения обратной связи.

---

## 1. Цветовая палитра (Color Palette)

Мы используем современную, мягкую и чистую цветовую палитру, вызывающую доверие и ассоциации с медициной и здоровьем.

### Основные цвета (Brand Colors)

| Название | Переменная CSS | HEX | Описание |
|----------|---------------|-----|----------|
| **Primary** | `--color-primary` | `#2E7D32` | Основной брендовый зеленый. Используется для кнопок, иконок и акцентов. |
| **Primary Hover** | `--color-primary-hover` | `#256628` | Темный оттенок зелёного для состояния наведения (hover). |
| **Secondary** | `--color-secondary` | `#E8F5E9` | Светло-зеленый фон. Используется для вторичных кнопок и фонов активных элементов. |
| **Secondary Hover** | - | `#DCEDC8` | Затемненный светло-зеленый для ховера вторичных кнопок. |

### Фоновые цвета (Backgrounds)

| Название | Переменная CSS | HEX | Описание |
|----------|---------------|-----|----------|
| **Background Body** | `--color-bg` | `#FAFAFA` | Общий фон страницы (Soft White/Grey). |
| **Background White** | `--color-bg-white` | `#FFFFFF` | Фон контейнеров и карточек. |

### Текст и Границы (Text & Borders)

| Название | Переменная CSS | HEX | Описание |
|----------|---------------|-----|----------|

|----------|---------------|-----|----------|

## 2. Типографика (Typography)

Основной шрифт интерфейса: **'Inter', sans-serif**.

| Элемент | CSS Класс / Селектор | Размер (Size) | Вес (Weight) | Цвет | Комментарий |
|---------|---------------------|---------------|--------------|------|-------------|
| **Заголовок страницы** | `.title` | `1.75rem` (28px) | 800 (ExtraBold) | Text Main | Основной заголовок экрана. |
| **Подзаголовок** | `.subtitle` | `1rem` (16px) | 400 (Regular) | Text Secondary | Поясняющий текст под заголовком. |
| **Заголовок формы** | `.form-title` | `1.5rem` (24px) | 700 (Bold) | Text Main | Заголовок конкретной формы. |
| **Label поля** | `.form-label` | `0.9rem` (14.4px) | 600 (SemiBold) | Text Main | Подпись к полю ввода. |
| **Основной текст** | `body` | `16px` | 400 (Regular) | Text Main | Базовый текст. |
| **Мелкий текст/Ошибки**| `.form-error`, `.form-hint` | `0.85rem` (13.6px)| 400 (Regular) | Error / Secondary | Подсказки и сообщения валидации. |

## 3. Тени и Скругления (Shadows & Radius)

Интерфейс использует мягкие тени для создания глубины и скругленные углы для дружелюбного вида.

### Тени (Shadows)

*   **Small (`--shadow-sm`)**: `0 1px 2px 0 rgba(0, 0, 0, 0.05)` — Для карточек и кнопок в покое.
*   **Medium (`--shadow-md`)**: `0 4px 6px -1px rgba(0, 0, 0, 0.1), ...` — Для ховера и акцентных элементов.
*   **Large (`--shadow-lg`)**: `0 10px 15px -3px rgba(0, 0, 0, 0.1), ...` — Для модальных окон и основного контейнера на десктопе.

### Скругления (Border Radius)

*   **Small (`--radius-sm`)**: `8px` — Для мелких элементов, тегов.
*   **Medium (`--radius-md`)**: `12px` — Для кнопок, полей ввода.
*   **Large (`--radius-lg`)**: `24px` — Для основного контейнера приложения.

## 4. Компоненты (Components)

### Кнопки (Buttons)

Все кнопки имеют базовый класс `.btn`. Стандартная высота: `56px`.

*   **Primary (`.btn-primary`)**: Зеленый фон, белый текст. Используется для основных действий (Отправить, ОК).
    *   *State Hover*: Затемнение фона + Тень Medium.
    *   *State Active*: Scale 0.98.
*   **Secondary (`.btn-secondary`)**: Светло-зеленый фон, темно-зеленый текст. Используется для навигации на главном экране.
    *   *State Hover*: Фон `#DCEDC8`.
*   **Outline (`.btn-outline`)**: Прозрачный фон, серая обводка (или цветная).
*   **Back Button (`.btn-back`)**: Текстовая кнопка со стрелкой "Назад". Без фона.

### Поля ввода (Inputs)

Базовый класс: `.form-input`.
*   **Default**: Фон `--color-bg`, граница прозрачная.
*   **Hover**: Фон `#F3F4F6`.
*   **Focus**: Белый фон, Граница Primary, Box-shadow Primary (с прозрачностью).
*   **Error**: Фон `#FEF2F2`, Граница Error, Текст Error.

### Чекбоксы (Checkboxes)

Кастомный CSS чекбокс `.checkbox-custom`.
*   **Unchecked**: Фон Body, Граница Border.
*   **Checked**: Фон Primary, Граница Primary, Белая галочка.

### Загрузка файла (File Upload)

Элемент `.btn-file-trigger`.
*   **Вид**: Dashed border (пунктирная граница), иконка скрепки.
*   **Hover**: Граница Primary, Фон Secondary.
*   **Состояние с файлом**: Сплошная граница Primary, Фон Secondary.
*   **Информация о файле (`.file-info`)**: Блок с именем файла и кнопкой удаления.

### Модальные окна (Modals)

*   **Overlay**: `rgba(0, 0, 0, 0.4)` с блюром (`backdrop-filter: blur(4px)`).
*   **Анимация**: `fadeIn` (появление) и `scaleIn` (увеличение контента).
*   **Контент**: Белый фон, скругление 20px, тень Large.

---

## 5. Анимации (Animations)

*   **Transition**: `0.2s cubic-bezier(0.4, 0, 0.2, 1)` — стандартная плавность для всех интерактивных элементов.
*   **Fade In**: Плавное появление экранов (opacity 0 -> 1, translateY 10px -> 0).
*   **Loader Spin**: Вращение индикатора загрузки на кнопке.

---

## 6. Адаптивность (Responsiveness)

*   **Mobile First**: Основные стили написаны для мобильных устройств.
    *   Контейнер на всю ширину и высоту экрана.
    *   Кнопки крупные, удобные для нажатия пальцем.
*   **Desktop (`min-width: 480px`)**:
    *   Контейнер центрируется, получает `max-width: 480px`.
    *   Появляются отступы от краев экрана, тень и скругленные углы у основного "телефона/карточки".
    *   Фон страницы (`body`) становится светло-серым `#F3F4F6`, чтобы выделить контент.
